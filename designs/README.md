# Codypool Design System

## 개요

Codypool은 AI 기반 패션 추천 서비스를 위한 통합 디자인 시스템입니다. MZ 세대를 타겟으로 하는 모던하고 직관적인 사용자 경험을 제공합니다.

## 디자인 시스템 구조

### 1. Design Tokens (`design.json`)

모든 시각적 요소는 중앙화된 디자인 토큰을 사용합니다:

- **색상 시스템**: Primary, Secondary, Accent, Success, Warning, Error, Neutral 팔레트
- **타이포그래피**: Noto Sans KR 기반 폰트 시스템
- **간격 시스템**: 8px 기반 배수 시스템
- **컴포넌트**: 버튼, 카드, 입력 필드 등의 재사용 가능한 컴포넌트
- **이미지 비율**: 웹사이트용 표준화된 이미지 비율 정의

### 2. 핵심 원칙

- **Token-First**: 모든 시각적 요소는 design.json 토큰 사용
- **일관성**: 정의된 디자인 시스템 준수
- **확장성**: 새로운 기능 추가 시 기존 시스템과 조화
- **접근성**: WCAG 2.1 AA 기준 준수

## 생성된 UI 페이지

### ui_1.html - AI 스타일 추천 튜토리얼

**목적**: 사용자에게 AI 스타일 추천 서비스의 사용법을 안내하는 인터랙티브 튜토리얼

**주요 기능**:
- 3단계 스와이프 가능한 튜토리얼 카드
- 키보드 및 터치 네비게이션 지원
- 반응형 디자인 (모바일 우선)
- 접근성 고려 (키보드 네비게이션, 스크린 리더 지원)

**디자인 특징**:
- MZ 세대 감성의 모던한 UI
- 부드러운 애니메이션과 전환 효과
- 직관적인 사용자 인터페이스
- 브랜드 컬러와 일관된 디자인

**기술 스택**:
- HTML5 (시맨틱 마크업)
- CSS3 (CSS 변수, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- 반응형 디자인


## 사용 가이드

### 1. 디자인 토큰 사용

```css
/* 올바른 사용법 */
.button {
  background: var(--color-primary-600);
  padding: var(--spacing-md) var(--spacing-xl);
  border-radius: var(--border-radius-lg);
}

/* 잘못된 사용법 - 하드코딩 금지 */
.button {
  background: #0284c7;
  padding: 16px 32px;
  border-radius: 8px;
}
```

### 2. 새로운 UI 페이지 생성

1. **요구사항 분석**: 페이지 목적과 타겟 사용자 정의
2. **디자인 토큰 적용**: design.json의 토큰 활용 계획
3. **HTML 구조 설계**: 시맨틱하고 접근성 높은 마크업
4. **CSS 스타일 구현**: 디자인 토큰 기반 스타일링
5. **JavaScript 기능 구현**: 모듈화된 인터랙션 로직
6. **품질 검증**: 체크리스트 통과 확인
7. **Sitemap 업데이트**: 페이지 정보 등록

### 3. 이미지 비율 가이드

- **1:1 (800x800)**: 썸네일, 아이콘, 프로필 이미지
- **16:9 (1280x720)**: 메인 배너, 비디오 커버
- **4:3 (1024x768)**: 콘텐츠 이미지, 갤러리 썸네일
- **3:4 (900x1200)**: 포스터형 이미지, 이벤트 카드
- **9:16 (720x1280)**: 모바일 전용 콘텐츠
